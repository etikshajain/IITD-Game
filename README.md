# Running the server
- To run the server on localhost
```
./scripts/run_server.sh
```
- To run the server on the wifi network
```
./scripts/run_server.sh wifi
```

# Joining the game
- On localhost
```
./scripts/run_game.sh
```
- On the wifi network
```
./scripts/run_game.sh wifi
```
